# Spring Boot Java8 WebApplication
Spring Boot Java8 QE2WebApplication
==================
Webapplication using Java8, Springboot's  MVC, JPA, JMS, SOAP Client/Servier, REST Client/Server and Maven


To Run
-------
Run using Spring tool suite - spring boot built in tomcat
http://localhost:2211/VehicleRegistrationView

Detailed explanation can be found in the resources/documents/ SpringbootQE2WebApplication.doc

Technologies
---------
- Java 8/J2E
- Spring Boot
- Spring MVC
- Spring JPA/Hibernate 
- Spring JMS/ActiveMQ
- Maven
- MySql
- Tomcat(Built In)
- JSP/JSTL
- HTML/CSS 
- Javascript/jQuery 
- SOAP Client/Server 
- REST Client/Server
